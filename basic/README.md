# 前端基础知识

本目录包含前端开发的基础知识，涵盖HTML、CSS、JavaScript等核心技术的入门和进阶内容。

## 目录结构

- [accessibility/](accessibility/README.md) - 无障碍设计与实现
- [css/](css/README.md) - CSS样式技术
- [html/](html/README.md) - HTML基础与语义化
- [js/](js/README.md) - JavaScript语言核心
- [performance/](performance/README.md) - 前端性能优化基础
- [security/](security/README.md) - 前端安全知识
- [testing/](testing/README.md) - 前端测试基础
- [ux/](ux/README.md) - 用户体验设计原则

## 学习路径

### 基础入门（推荐顺序）

1. **HTML基础** - 掌握网页结构和语义化标签
2. **CSS基础** - 学习样式设置和布局技术
3. **JavaScript基础** - 理解编程语言核心概念

### 进阶学习

1. **JavaScript ES6+** - 现代JavaScript特性
2. **CSS进阶** - 动画、响应式设计、预处理器
3. **性能优化基础** - 加载优化、渲染优化
4. **前端安全** - XSS防护、CSRF防护等

### 专业提升

1. **无障碍设计** - 确保网站对所有人可用
2. **用户体验设计** - 提升产品的可用性和满意度
3. **前端测试** - 单元测试、集成测试基础

## 学习建议

1. **循序渐进** - 从基础到进阶，逐步深入
2. **实践为主** - 每个概念都通过实际代码练习巩固
3. **查阅规范** - 参考官方文档和行业标准
4. **关注兼容性** - 了解不同浏览器的支持情况
5. **持续学习** - 前端技术快速发展，保持学习状态
