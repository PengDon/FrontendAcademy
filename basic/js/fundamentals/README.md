# JavaScript 基础知识点

## 核心概念

### 变量声明
```javascript
// var - 函数作用域，可重复声明
var name = 'John';

// let - 块级作用域，不可重复声明
let age = 30;

// const - 块级作用域，不可重新赋值（但对象属性可以修改）
const PI = 3.14159;
```

### 数据类型

**基本数据类型：**
- Number: 数字
- String: 字符串
- Boolean: 布尔值(true/false)
- Null: 空值
- Undefined: 未定义
- Symbol: 唯一标识符（ES6新增）
- BigInt: 大整数（ES2020新增）

**引用数据类型：**
- Object: 对象
- Array: 数组
- Function: 函数
- Date: 日期
- RegExp: 正则表达式

### 类型转换

**显式转换：**
```javascript
// 转为字符串
String(123);  // "123"

// 转为数字
Number('123');  // 123
parseInt('123abc');  // 123
parseFloat('123.45abc');  // 123.45

// 转为布尔值
Boolean('hello');  // true
Boolean('');  // false
Boolean(0);  // false
Boolean(null);  // false
Boolean(undefined);  // false
```

**隐式转换：**
```javascript
// 字符串连接
'10' + 5;  // "105"

// 数字运算
'10' - 5;  // 5

// 布尔运算
!!'hello';  // true
```

## 函数

### 函数声明
```javascript
function add(a, b) {
    return a + b;
}
```

### 函数表达式
```javascript
const add = function(a, b) {
    return a + b;
};
```

### 箭头函数（ES6）
```javascript
const add = (a, b) => a + b;
```

## 常见问题与答案

### 1. var、let和const的区别是什么？
**答案：**
- **var**：函数作用域，可以重复声明，存在变量提升
- **let**：块级作用域，不能重复声明，不存在变量提升，暂时性死区
- **const**：块级作用域，不能重复声明，不能重新赋值（但对象属性可以修改），不存在变量提升，暂时性死区

### 2. == 和 === 的区别是什么？
**答案：**
- `==` 比较值是否相等，会进行类型转换
- `===` 比较值和类型是否都相等，不会进行类型转换

### 3. 什么是闭包？
**答案：** 闭包是指有权访问另一个函数作用域中变量的函数。闭包的作用包括：
- 可以访问局部变量
- 可以保存状态
- 可以实现模块化

### 4. 什么是原型和原型链？
**答案：**
- 原型是JavaScript中对象的一个属性，每个对象都有一个原型
- 原型链是指对象查找属性或方法时，如果在自身找不到，就会去原型对象中查找，如果原型对象中也找不到，就会去原型的原型中查找，直到找到Object.prototype

### 5. this指向问题
**答案：** this的指向取决于函数的调用方式：
- 作为对象方法调用时，this指向该对象
- 作为普通函数调用时，this指向全局对象（浏览器中是window）
- 作为构造函数调用时，this指向新创建的对象
- 使用call/apply/bind时，this指向指定的对象
- 箭头函数中的this继承自外层作用域

## 最佳实践

1. **使用严格模式**：在脚本开头添加`'use strict';`
2. **使用const和let**：避免使用var
3. **使用箭头函数**：在合适的场景使用箭头函数简化代码
4. **避免使用全局变量**：尽量使用局部变量
5. **使用模板字符串**：使用反引号和${}进行字符串拼接
6. **使用解构赋值**：简化对象和数组的访问
7. **使用默认参数**：为函数参数提供默认值
8. **使用数组方法**：如map、filter、reduce等函数式方法
9. **添加适当的注释**：解释复杂的逻辑和功能
