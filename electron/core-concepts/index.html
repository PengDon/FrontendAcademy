Electron 核心概念示例

Electron 介绍

Electron 是一个使用 JavaScript、HTML 和 CSS 构建跨平台桌面应用程序的框架。它结合了 Chromium、Node.js 和操作系统特定的 API，使开发者能够使用 Web 技术创建在 Windows、macOS 和 Linux 上运行的桌面应用。

注意：这个页面正在渲染进程中运行，通过预加载脚本与主进程通信。

进程模型

Electron 应用由以下主要进程组成：

主进程 (Main Process)

  • 每个应用只有一个主进程
  • 负责创建和管理浏览器窗口
  • 处理系统事件和原生功能
  • 入口点是 package.json 中指定的 main 文件

渲染进程 (Renderer Process)

  • 每个浏览器窗口运行一个渲染进程
  • 负责渲染页面内容（HTML、CSS、JavaScript）
  • 默认情况下，渲染进程没有 Node.js 功能的访问权限

预加载脚本 (Preload Scripts)

  • 在渲染进程加载网页之前执行
  • 可以访问 Node.js API 和 Electron API
  • 用于在渲染进程和主进程之间安全地暴露 API
