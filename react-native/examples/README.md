# React Native 示例项目

本目录包含React Native的示例项目，帮助开发者通过实际项目学习React Native的使用。

## 内容概述

- **todo-app/**: 待办事项应用，展示基础组件和状态管理
- **weather-app/**: 天气应用，展示网络请求和API集成
- **e-commerce/**: 电商应用，展示复杂布局和导航
- **social-media/**: 社交媒体应用，展示用户界面和交互设计

## 学习重点

1. 通过实际项目学习React Native的应用
2. 了解不同类型应用的实现方式
3. 掌握项目结构和代码组织
4. 学习最佳实践和设计模式

## 示例项目结构

```
todo-app/
├── src/
│   ├── components/        # 自定义组件
│   ├── screens/          # 页面组件
│   ├── hooks/            # 自定义Hooks
│   ├── context/          # Context API
│   ├── utils/            # 工具函数
│   └── types/            # TypeScript类型定义
├── App.tsx               # 应用入口
├── package.json          # 项目配置
└── tsconfig.json         # TypeScript配置
```

## 如何运行示例项目

1. 进入示例项目目录
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm start`
4. 使用Expo Go应用扫描二维码，或运行在模拟器中
