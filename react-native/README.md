# React Native 复习知识

本目录包含React Native的核心知识点、组件开发、导航、状态管理等内容，帮助开发者系统复习和学习React Native。

## 目录结构

- **core-concepts/**: React Native核心概念
- **components/**: 组件开发
- **navigation/**: 导航系统
- **state-management/**: 状态管理
- **networking/**: 网络请求
- **styling/**: 样式系统
- **performance/**: 性能优化
- **testing/**: 测试
- **examples/**: 示例项目
- **interview/**: 面试题

## 学习路径

1. 先学习**core-concepts**掌握React Native基础
2. 然后学习**components**和**styling**进行UI开发
3. 接着学习**navigation**实现页面跳转
4. 再学习**state-management**管理应用状态
5. 最后学习**networking**、**performance**和**testing**等进阶内容
