# Flutter 示例项目

本目录包含完整的Flutter示例应用，帮助您学习如何将Flutter的各种功能整合到实际项目中。每个示例项目都针对特定的学习目标或应用场景进行设计。

## 目录结构

```
examples/
├── todo_app/              # 待办事项应用（基础状态管理）
├── weather_app/           # 天气预报应用（网络请求和状态管理）
├── social_media_demo/     # 社交媒体演示应用（复杂UI和交互）
├── e_commerce_app/        # 电子商务应用（数据持久化和路由）
├── animation_gallery/     # 动画展示应用（各种动画效果）
└── expense_tracker/       # 支出跟踪应用（数据可视化）
```

## 示例项目介绍

### 1. Todo App - 待办事项应用

一个简单但功能完整的待办事项应用，展示了如何实现基础的状态管理和用户交互。

**核心功能：**
- 添加、编辑、删除待办事项
- 标记任务完成/未完成
- 按类别筛选任务
- 简单的本地数据存储

**学习要点：**
- 使用setState管理本地状态
- 使用Listview构建动态列表
- 表单处理和用户输入验证
- SharedPreferences实现简单数据持久化

### 2. Weather App - 天气预报应用

一个完整的天气预报应用，展示了如何从API获取数据并在应用中展示。

**核心功能：**
- 搜索和选择城市
- 显示当前天气和未来几天预报
- 展示天气详情（温度、湿度、风速等）
- 根据天气状况显示不同的背景和图标

**学习要点：**
- 使用http或dio包进行网络请求
- JSON数据解析和模型转换
- 使用Provider管理应用状态
- 处理异步操作和错误
- 响应式UI设计

### 3. Social Media Demo - 社交媒体演示应用

一个简化的社交媒体应用，展示了如何构建复杂的UI和处理各种用户交互。

**核心功能：**
- 用户认证（登录/注册）
- 浏览和发布帖子
- 点赞和评论功能
- 个人资料页面

**学习要点：**
- 复杂的UI布局和组件嵌套
- 自定义组件和主题
- 页面转场和动画效果
- 用户认证流程
- 社交互动功能实现

### 4. E-commerce App - 电子商务应用

一个电子商务应用，展示了产品展示、购物车和结账流程。

**核心功能：**
- 产品列表和详情页
- 购物车功能
- 分类浏览
- 本地收藏和购物车数据持久化

**学习要点：**
- 复杂导航结构
- 数据持久化（SQLite或Hive）
- 购物车状态管理
- 产品搜索和筛选
- 响应式布局适配不同屏幕

### 5. Animation Gallery - 动画展示应用

一个专注于展示各种Flutter动画效果的应用。

**核心功能：**
- 展示各种基本动画（淡入淡出、旋转、缩放等）
- 复杂动画效果（交错动画、物理动画等）
- 交互式动画示例
- 性能优化技巧演示

**学习要点：**
- 基础动画API使用
- 自定义动画控制器
- 性能优化策略
- 动画组合和序列
- 响应式动画设计

### 6. Expense Tracker - 支出跟踪应用

一个个人财务管理应用，展示了数据可视化和复杂的用户输入处理。

**核心功能：**
- 添加和分类支出
- 支出统计和图表展示
- 设置预算和提醒
- 导出数据报告

**学习要点：**
- 数据可视化（使用charts_flutter等库）
- 表单验证和复杂用户输入
- 本地数据库操作
- 日期和时间处理
- 数据导出和共享

## 如何使用示例项目

1. 克隆或下载本仓库
2. 进入相应的示例项目目录
3. 运行`flutter pub get`安装依赖
4. 运行`flutter run`启动应用

## 学习路径建议

建议按照以下顺序学习示例项目：

1. 从**Todo App**开始，学习基础Flutter概念和状态管理
2. 学习**Weather App**，掌握网络请求和更复杂的状态管理
3. 探索**Animation Gallery**，了解Flutter的动画系统
4. 研究**Social Media Demo**和**E-commerce App**，学习复杂UI和交互设计
5. 最后学习**Expense Tracker**，掌握数据可视化和高级功能
