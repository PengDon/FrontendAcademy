# FrontendAcademy

## 项目介绍

本项目是一个全面的前端学习知识库，用于复习前端知识点，应对面试和回顾核心概念。项目覆盖从基础的HTML/CSS/JavaScript到高级前端技术、框架生态、算法数据结构以及Node.js后端开发。

## 目录说明

### basic/
包含前端开发的基础知识，分为HTML、CSS、JavaScript三个主要部分，以及无障碍访问、性能优化基础、安全基础、测试基础和用户体验等相关知识，适合初学者和需要复习基础的开发者。

### algorithm-data-structure/
覆盖常见的数据结构（数组、链表、树、图、哈希表）和算法（排序、搜索、动态规划、回溯），帮助提升编程能力和解决问题的思维方式。

### design-patterns/
包含设计模式的分类实现，按照创建型、结构型和行为型进行组织，提供可复用的设计解决方案。

### functional-programming/
介绍函数式编程的核心概念，如高阶函数、柯里化、函数组合、纯函数和不可变性。

### electron/
Electron桌面应用开发相关知识，包括核心概念、主进程与渲染进程通信、窗口管理、菜单对话框、打包发布和安全最佳实践等。

### flutter/
Flutter跨平台移动应用开发相关知识，包括核心概念、组件开发、状态管理、动画效果、网络请求、路由导航和数据持久化等。

### regex/
正则表达式相关知识，包括基本语法、元字符、量词、分组、断言和常见应用场景等。

### vue/
Vue.js生态系统相关知识，包括Vue 2、Vue 3（组合式API）、Nuxt.js服务端渲染框架、Vue Router路由管理和状态管理（Vuex/Pinia）等。

### react/
React生态系统相关知识，包括基础组件开发、Hooks使用、Redux状态管理、Next.js框架、React Router和Context API等。

### advanced/
高级前端概念，包括微前端架构、性能优化与监控、PWA、WebAssembly、SSR、测试工具、构建工具、TypeScript、浏览器插件开发、安全和可视化技术（Three.js、WebGL、ECharts）等。

### nodejs/
Node.js后端开发相关内容，包括核心模块、Express和Nest.js框架、各种数据库连接、认证授权、API设计（RESTful和GraphQL）、微服务架构和Web框架等。

## 使用方法

1. 根据需要复习的知识点，进入对应的目录查看示例代码和笔记
2. 每个目录下可以添加示例代码、文档说明或练习题目
3. 推荐按照从基础到高级的顺序进行学习和复习

## 学习建议

- 基础部分：确保掌握HTML语义化、CSS布局技术和JavaScript核心概念
- 算法与数据结构：重点掌握常见的数据结构实现和算法思想
- 框架学习：先掌握Vue或React其中一个框架，再学习另一个
- 高级概念：在掌握基础和框架后，再深入学习高级主题
- 实战练习：结合每个目录的知识点，编写小型项目巩固所学内容

## 面试准备

针对面试，建议重点关注以下内容：
- JavaScript核心概念（闭包、原型链、异步编程）
- 框架原理（虚拟DOM、组件生命周期、响应式系统）
- 性能优化（渲染性能、网络优化、构建优化）
- 算法与数据结构（常见排序算法、树操作、动态规划）
- 设计模式（在前端中的应用场景）
